# 💬 ChatTalk | 실시간 그룹 채팅 서비스
실시간 소셜 그룹 채팅 플랫폼 ｜ WebSocket & Redis Pub/Sub 기반 메시징 ｜ AWS & Kubernetes 기반 인프라 구축

---

## 💡 프로젝트 개요
> 사용자 간 친구를 맺고 채팅방을 만들어 대화를 나눌 수 있는 SNS 기반 실시간 채팅 서비스

- JWT 인증 처리 및 권한 관리
- 친구 추가 / 채팅방 생성 / 실시간 채팅
- 과거 채팅 내역 DB 저장 및 조회
- WebSocket + Redis Pub/Sub 기반 메시징 처리
- AWS + Docker + K3s 기반 배포
- GitHub Actions 기반 CI/CD 자동화 구성
